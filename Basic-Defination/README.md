## Data:

### Example:

## Data structure:
The logical or mathematical model of a particular organization of data is called data structure.
(-  ডাটা স্ট্রাকচার হলো ডেটাকে এমনভাবে সাজানোর কৌশল যাতে তা দ্রুত এবং কার্যকরভাবে ব্যবহার করা যায়।)

## Premitive data:
(also called simple data type)
Basic data type that consist integer, real, character, and boolean that cannot be devided called primitive data type.

## Non-primitive data:
The data structure witch is processing of complex number is called non-primitive data type.

# Primitive vs Non-Primitive Data Structures

| Aspect        | Primitive Data Structure | Non-Primitive Data Structure |
|---------------|--------------------------|------------------------------|
| **Complexity** | Simple, basic            | Complex, built from primitives |
| **Examples**   | int, float, char, boolean | Array, Linked List, Stack, Queue, Tree, Graph |
| **Size**       | Fixed                    | Dynamic / flexible |
| **Usage**      | Direct data storage       | Organizing and managing collections of data |
| **Operations** | Direct machine-level operations | Require algorithms for manipulation |
